### 4. Setting

- eslint & prettier

  ```javascript
  npm i -D eslint
  npm i -D eslint-config-airbnb-typescript eslint-plugin-import
  npm i -D prettier eslint-config-prettier
  ```

- typescript

  ```javascript
  npm i -D typescript
  npm i -D @typescript-eslint/parser @typescript-eslint/eslint-plugin
  ```

- lite-server

  ```javascript
  npm i -D lite-server
  ```

- concurrently

  ```javascript
  npm i -D concurrently
  ```
